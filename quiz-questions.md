# The Depleted Dad Diagnostic — Quiz Questions (v2.1)

11 questions. Each option maps to a depletion type and a point value. Every question except Q8 also has a **managing** option that scores 0 — it never produces a result type, it only pulls the severity total down.

**Scoring:** each answer adds its points to that depletion type. Highest total wins; ties break by most-selected type, then priority order **bouncer → ghost → sleeper → engine**. (Severity is the sum of all type points — see `quiz-scoring.md`.)

**Result types:** the Ghost (identity depletion), the Engine (output depletion), the Bouncer (regulation depletion), the Sleeper (physical depletion).

Source: `src/lib/quiz-data.ts`
