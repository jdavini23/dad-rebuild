# The Depleted Dad Diagnostic — Quiz Questions

12 questions. Each option maps to a depletion type and a point value.

**Scoring:** each answer adds its points to that depletion type. Highest total wins; ties break in priority order **bouncer → ghost → sleeper → engine**.

**Result types:** Ghost (identity depletion), Engine (output depletion), Bouncer (regulation depletion), Sleeper (physical depletion).

Source: `src/lib/quiz-data.ts`

---
